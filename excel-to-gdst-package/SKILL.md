# Excel to GDST Converter Skill

## 概述
将 Excel 规则表转换为 GDST（决策表）文件的自动化工具。

## 使用场景
- 从 Excel 表格生成 GDST 决策表文件
- 支持动态表头识别
- 支持多条件列的规则配置

## 调用方式
```bash
python excel_to_gdst_v2.py [Excel文件路径]
```

## Excel 格式要求

### 表头结构
| 列位置 | 内容 | 说明 |
|--------|------|------|
| 第1行 | 元数据 | 可选，包含描述信息 |
| 第2行 | 表头 | 必须包含"规则名"和"结果" |

### 表头列说明
- **规则名**: 规则标识列（必需）
- **条件**: 规则条件表达式
- **其他条件列**: 如"车分层"、"蚂蚁星河标签"等（可扩展）
- **结果**: 规则执行结果（必需）

### 数据格式
```
描述 | 规则名 | 条件 | 车分层 | 蚂蚁星河标签 | 结果
A-1234-1 | myxh_AmtIndex_A | v.get("risk_group")=="A" | | v.get("myxh_risk_seg")==1 | myxh_amtindex_a_jixin=1.2
```

## 输出说明
- **文件名**: 取第一个条件列的值作为表名
- **格式**: GDST XML 格式
- **编码**: UTF-8

## 特性
1. **动态表头识别**: 自动识别 Excel 中的条件列
2. **智能描述提取**: 从规则名列之前的列获取描述
3. **结果格式化**: 自动处理结果值格式
4. **多 Sheet 支持**: 处理 Excel 中的多个工作表

## 示例

### 输入 Excel
```
F | 描述 | f:CommonRuleFact | | | | f
| | 规则名 | 条件 | 车分层 | 蚂蚁星河标签 | 结果
| A-1234-1 | myxh_AmtIndex_A | v.get("risk_group")=="A" | | v.get("myxh_risk_seg")==1 | myxh_amtindex_a_jixin=1.2
```

### 输出 GDST
- 表名: myxh_AmtIndex_A
- 条件列: 规则名、条件、车分层、蚂蚁星河标签
- 动作列: 结果
- 数据行: 按 Excel 数据生成

## 注意事项
1. Excel 文件的第一行可以是元数据行
2. 表头行必须包含"规则名"和"结果"
3. 描述信息从规则名列之前的列提取
4. 结果列如果没有"="符号，会自动添加默认权重"=60"
5. 特殊字符会自动转义（XML 转义）

## 错误处理
- 如果找不到"规则名"或"结果"列，会跳过该 Sheet
- 如果数据行没有结果值，会跳过该行
- 支持部分空值的处理
